Description

Welcome to the site of Real Survival. Real survival is a survival game which aims to be as realistic as possible. It uses concepts such as hunger and thirst which can eventually cause death. Time ticks and you keep getting hungrier and thirstier over time.
The goal of Real Survival is simple: Survive!

Trailer

Controls

Walking - 'W'(forward) 'A'(left) 'S'(back) 'D'(right)
Jumping - 'Space'
Sprint - 'SHIFT'
Pick up item - Left click
Lay down item - Right click
Eat & Drink - 'E'
Pauze - 'ESC'
